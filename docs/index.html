ActionHero

The reusable, scalable, and quick node.js API server for stateless and stateful applications

NPM | Docs | Web Site | GitHub | Slack | Client

Who is the ActionHero?

ActionHero is a multi-transport API Server with integrated cluster capabilities and delayed tasks. The goal of actionhero is to create an easy-to-use toolkit for making reusable & scalable APIs for HTTP, WebSockets, and more. Clients connected to an actionhero server can consume the api, consume static content, and communicate with each other. ActionHero is cluster-ready, with built in support for background tasks, 0-downtime deploys, and more.

Currently actionhero supports the following out of the box...

  • Web Clients: HTTP, HTTPS
  • Socket Clients: TCP (telnet), TLS
  • Web Socket Clients: HTTP, HTTPS

... and you can also make your own servers and transports.

Quick Start

# mkdir new_project; cd new_project
